Sebelum menjalankan project ini, pastikan Python sudah terinstall pada komputer.

## 1. Cek apakah Python sudah terinstall
Buka Terminal / CMD lalu jalankan:
py --version


Jika muncul versi Python (contoh: Python 3.12.0), berarti Python sudah terpasang.
Jika belum ada, download dan install Python terlebih dahulu:
https://www.python.org/downloads/



## 2. Install Flask
Buka Terminal / CMD pada folder project, lalu jalankan:
py -m pip install flask


## 3. Menjalankan aplikasi
Masuk ke folder project lalu jalankan:
py app.py

Setelah berhasil dijalankan, buka:
http://127.0.0.1:5000
